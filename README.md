<ol>
  <li>Задача №1: На какую сумму мы могли бы увеличить расходы на основных сотрудников (Fixed costs / ФОТ) в апреле 2021 года  при условии, что Маржинальность в этом месяце не должна упасть ниже 11%?</li>
